# Project Description

**Restaurant Panel** is a full-featured restaurant management system built to streamline day-to-day operations across multiple roles and departments. From the kitchen to the billing counter, from staff management to super admin oversight — every part of your restaurant workflow is covered in one unified platform.

Whether you're running a single outlet or a chain of restaurants, this system scales with your needs, offering real-time order tracking, AI-powered demand forecasting, inventory control, and multi-gateway payment support.


#  Panels & Features
#  Login Screen
- Secure Gmail + Password authentication
- New user registration support
- Clean dark-themed UI


#  Super Admin Panel
- Bird's-eye view of **all branches**
- Monitor **Total Sales**, **Active Branches**, **Orders Today**
- **Demand Forecast** across outlets
- **Inventory Alerts** for stock issues
- Quick Actions: Add Branch · View Reports · Manage Staff


# Menu & Category Management
- Add, edit, and delete **menu categories** (Starters, Main Course, Beverages, Desserts)
- Toggle category **Active / Inactive** status
- Add and manage **dishes** with image, price, and availability
- Search dishes by name or category
- Sort by price


# Inventory Management
- Track **Total Items**, **Low Stock Alerts**, **Categories**, and **Out of Stock** items
- Search inventory with filters by category and status
- **Add New Item** and **View Low Stock** shortcuts
- Real-time stock level monitoring


# Analytics
- Key metrics at a glance: **New Orders · Sales · Demand · Expenses**
- **Total Sales** line chart (monthly trend: Jan – Nov)
- **Top Performing Dishes** section
- **Demand Prediction** module for forecasting


# Billing & Payments
- Configure **Tax (%)**, **Discount (%)**, and **Service Charge (%)**
- Payment gateway integrations:
  - **Razorpay API**
  - **Paytm Merchant**
  - **UPI ID**
- Supported payment modes: **Cash · UPI · Card**


# Kitchen Dashboard
- Real-time order board with three columns:
  - 🟡 **New Order** — Incoming orders with item list and special instructions
  - 🔵 **In Progress** — Orders currently being prepared
  - 🟢 **Ready to Serve** — Completed orders
- Order details include: Order No., Table No., Items & Quantity, Notes (e.g., "Spicy")
- One-click **START** and **COMPLETE** actions


# Staff Panel (Order Entry)
- Staff can **select table** and build orders
- Add items with quantity and price
- View itemized order list
- **Proceed to Billing** with one click
